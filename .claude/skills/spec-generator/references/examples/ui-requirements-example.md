# UI Requirements 生成例（タスク管理機能）

## コンポーネント階層

```
+page.svelte (タスク一覧画面)
├── TaskList.svelte
│   └── TaskItem.svelte (複数)
├── TaskCreateDialog.svelte
└── TaskFilterBar.svelte
```

## レイアウト・スペーシング

| 要素         | 配置         | 幅        | 余白                   |
| ------------ | ------------ | --------- | ---------------------- |
| ヘッダー     | 固定         | w-full    | p-4                    |
| フィルタバー | 固定         | w-full    | p-4 pb-2               |
| タスクリスト | スクロール   | w-full    | px-4 pb-4              |
| FABボタン    | 固定（右下） | w-14 h-14 | fixed bottom-4 right-4 |

## 状態ごとの表示ルール

| 要素           | 条件            | 表示                                       | 備考                                      |
| -------------- | --------------- | ------------------------------------------ | ----------------------------------------- |
| CompleteButton | 未完了          | 表示                                       |                                           |
| CompleteButton | 完了済み        | 非表示（DOM除去）                          |                                           |
| DeleteButton   | status=active   | 表示                                       |                                           |
| DeleteButton   | status=archived | disabled（opacity-50, cursor-not-allowed） | tooltip「アーカイブ済みは削除できません」 |

## インタラクション

| 操作                     | 結果                         |
| ------------------------ | ---------------------------- |
| タスク名クリック         | 詳細ダイアログを開く         |
| チェックボックスクリック | タスクの完了状態を切り替える |
| FABボタンクリック        | 作成ダイアログを開く         |

## アニメーション・トランジション

| 要素                 | トリガー  | アニメーション種別 |
| -------------------- | --------- | ------------------ |
| ダイアログ           | 開閉      | fade-in/scale-in   |
| タスクリストアイテム | 追加/削除 | slide-in/slide-out |

## レスポンシブ挙動

| 要素         | sm (640px)   | md (768px)   | lg (1024px) |
| ------------ | ------------ | ------------ | ----------- |
| ヘッダー     | h-16         | h-16         | 変更なし    |
| フィルタバー | 横スクロール | 横スクロール | 変更なし    |
| タスクリスト | 1列          | 1列          | 2列（grid） |

## バリデーション表示

| フィールド | エラー条件 | 表示方法                   |
| ---------- | ---------- | -------------------------- |
| タイトル   | 空文字     | 入力欄の直下に赤色テキスト |
| 期限日     | 過去の日付 | 入力欄の直下に赤色テキスト |

## 空状態・ローディング

| 状態       | 表示内容                                                                   |
| ---------- | -------------------------------------------------------------------------- |
| 0件        | 「タスクがありません。FABボタンから作成してください」メッセージ + アイコン |
| 読み込み中 | スピナー表示                                                               |
