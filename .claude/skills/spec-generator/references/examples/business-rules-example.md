# Business Rules 生成例（支出承認機能）

## 状態遷移

```
pending ──approve──→ approved
approved ──reject──→ pending
```

## 権限モデル

| 操作                | 申請者           | 承認者 | 管理者 |
| ------------------- | ---------------- | ------ | ------ |
| 作成                | ○                | ○      | ○      |
| 更新（pendingのみ） | ○                | ×      | ○      |
| 承認                | ×                | ○      | ○      |
| 却下                | ×                | ○      | ○      |
| 削除                | ○（pendingのみ） | ×      | ○      |
